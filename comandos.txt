
$ git config --global ---> para administrar las configuraciones globales y no locales

$ git config --global user.name ---> para administrar el nombre como te vas a identificar

$ git config --global user.email "correo@correo.com" ---> para configurar el correo con el que te vas a identificar


$ git config --global --list ---> configuraciones globales

$ git config --global core.editor "code --wait" ---> para configurar visual estudio code como IDE predeterminado


$ git config --global color.ui true --> para cofigurar color de tu IDE
 
$ git config --global autocrlf true ---> (solo para windows) para hacer un salto de linea y retornar al principio.
